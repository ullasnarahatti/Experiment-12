# Cascode Amplifier with Active Load -- LTspice Analysis (Circuit 2B)

## Circuit Overview

This project analyzes a **Common Source Cascode Amplifier with an Active
Load** using LTspice.\
The cascode structure improves amplifier gain and bandwidth by
increasing output resistance and reducing the Miller effect.

![Circuit](https://github.com/user-attachments/assets/28df0e80-eac5-4c82-9b43-512590f96609)

# 1. DC Operating Conditions

  Parameter                Value
  ------------------------ ---------
 - Supply Voltage (VDD)     1.5 V
 - Desired Drain Current    200 µA
 - NMOS Threshold Voltage   0.36 V
 - PMOS Threshold Voltage   −0.39 V

These values determine the bias conditions required for proper
transistor operation.

------------------------------------------------------------------------

# 2. Bias Voltage Constraints

  Quantity   Minimum    Maximum   Purpose
  ---------- ---------- --------- ---------------------
 - NMOS VGS   ≥ 0.36 V   ≤ 1.5 V   Channel formation
 - NMOS VDS   ≥ VOV      ≤ VDD     Maintain saturation
 - PMOS VSG   ≥ 0.39 V   ≤ VDD     PMOS conduction
 - PMOS VSD   ≥ VOV      ≤ VDD     Maintain saturation

------------------------------------------------------------------------

# 3. Overdrive Voltage Calculation

The overdrive voltage determines the strength of the MOS transistor.

VGS = VTH + VOV\
VGS = 0.36 + 0.25 = **0.61 V**

VOV = VGS − VTH = **0.25 V**

Expected drain voltage:

VDS ≈ VDD / 2 ≈ **0.75 V**

Allowable range:

0 \< VOV \< 0.75 V

The selected value **0.25 V** satisfies this condition.

Reasons:

-   Provides adequate transconductance
-   Maintains stable current
-   Leaves enough voltage headroom for signal swing

------------------------------------------------------------------------

# 4. Intermediate Node Voltage (VS1)

In the cascode configuration, the **drain of the lower NMOS becomes the
source of the upper NMOS**.

To keep the lower transistor in saturation:

VDS2 ≥ VOV

Since the source is grounded:

VS2 = 0

Therefore:

VDS2 = VS1

Condition:

VS1 ≥ 0.25 V

Chosen value:

VS1 ≈ **0.3 V**

------------------------------------------------------------------------

# 5. Output Voltage

For symmetrical signal swing:

VDS ≈ VDD / 2 ≈ **0.75 V**

Output voltage:

Vout = VDS + VS1\
Vout = 0.75 + 0.3 = **1.15 V**

------------------------------------------------------------------------

# 6. Saturation Verification

  ------------ ----------- --------------
  M2 (NMOS)    VDS ≥ VOV   0.3 ≥ 0.25 ✔
  M1 (NMOS)    VDS ≥ VOV   0.9 ≥ 0.25 ✔
  M3 (PMOS)    VSD ≥ VOV   0.6 ≥ 0.25 ✔

# 8. Transistor Width Selection

The transistor widths were initially estimated using the **square‑law
MOSFET current equation** to obtain ID ≈ 200 µA.

  Transistor          Calculated Width   Adjusted Width
  ------------------- ------------------ ----------------
  M1 (NMOS Cascode)   15.16 µm           15.4 µm
  M2 (NMOS Input)     15.16 µm           15.4 µm
  M3 (PMOS Load)      35.9 µm            37 µm

The final values were slightly tuned in simulation due to:

-   Channel length modulation
-   Mobility degradation
-   Cascode bias sensitivity

# 10. Voltage Gain

Voltage gain:

Av = Vout / Vin

Measured values:

Av = (1.060 − 1.027) / (0.9195 − 0.9005)\
Av ≈ **1.72**

Gain in decibels:

Gain(dB) = 20 log10(Av)\
Gain ≈ **4.77 dB**

------------------------------------------------------------------------

# 13. Conclusion

The **Common Source Cascode Amplifier with Active Load** was
successfully designed and simulated in LTspice.

Key results:

-   Stable bias current close to **200 µA**
-   Voltage gain around **4.8 dB**
-   Wide bandwidth of approximately **270 MHz**

The cascode structure improves amplifier performance by increasing
output resistance and reducing capacitance effects, resulting in
improved gain and bandwidth compared to a simple common‑source
amplifier.
